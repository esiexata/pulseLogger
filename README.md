# Power meter logger
  ___                    
 | _ \_____ __ _____ _ _ 
 |  _/ _ \ V  V / -_| '_|
 |_| \___/\_/\_/\___|_|  
                                
A Raspberry pi reads power consumption from an electricity meter via the S0 led pulse output, and stores it in an influx database.

Sensor: Osram, SFH 3310 phototransistor
